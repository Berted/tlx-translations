Description

Pak Chanek is a math teacher. Today, he is teaching $N$ students to count. Pak Chanek will give his $i$-th student an interval of numbers $[L_i, R_i]$. Then, the student will take turns (from student $1$ to student $N$) counting all numbers on the interval $[L_i, R_i]$ that has not been counted by a previous student. For example, say student $1$ gets interval $[2, 5]$. Then, if the student $2$ gets interval $[1, 6]$, they will only count $1$ and $6$, because all numbers from $2$ to $5$ has been counted by student $1$.

After all of Pak Chanek's students have counted the numbers, Pak Chanek wonders what is the sum of all the numbers that his students counted. Help Pak Chanek satisfy his curiosity!

Input Format

The first line consists of a single integer $N$ - the number of Pak Chanek's students.

The next $N$ lines contains two integers $L_i$ and $R_i$ - the interval that the $i$-th student receives from Pak Chanek.

Output Format

An integer, the sum of all numbers that Pak Chanek's students counted.

Sample Input

2
3 5
4 6

Sample Output

18

Constraints

  • $1 \le N \le 100\,000$
  • $1 \le L_i \le R_i \le 10^9$

For all subtasks:

  • $1 \le N \le 100\,000$
  • $1 \le L_i \le R_i \le 10^9$

Subtask 1 (16 points)

  • $1 \le N \le 1\,000$
  • $1 \le L_i \le R_i \le 1\,000$

Subtask 2 (29 points)

  • $1 \le N \le 1\,000$

Subtask 3 (14 points)

  • $1 \le N \le 100\,000$
  • No $j$ exists such that $L_i \le L_j \le R_i$ for $1 \le i \le N$

Subtask 4 (20 points)

  • $1 \le N \le 100\,000$
  • $1 \le L_i \le R_i \le 100\,000$

Subtask 5 (21 points)

  • No additional constraints.
